Simple Tool that uses Mutagen to transfer Bethesda Mods to readable files

Currently Implemented:

- Skyrim to SQL DB
- Skyrim to csv files
- Skyrim to xlsx files
- crappy UI

Planned:

- Skyrim to Google Sheets
- Skyrim to xml
- Other Bethesda Mods to all types
- Good UI

How to use:

- go to latest Release and download attached .zip
- unzip and run .exe
- use of Sql Database requires you to have a sql server up and running
